DDDとはずばりソフトウェア設計手法の1つで、
「ある領域に存在する特定の問題を解決することにフォーカスした設計手法」と言えるでしょう。

ソフトウェアというのは何か問題を解決するために作られますが、
ソフトウェアで問題解決しようとする対象領域のことをドメインと呼びます。
